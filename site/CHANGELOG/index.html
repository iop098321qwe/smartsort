Changelog¶

All notable changes to this project will be documented in this file. See commit-and-tag-version for commit guidelines.

0.1.1 (2026-02-28)¶

Features¶

  • undo: increase default history depth to 10 (2cdc972)
  • undo: store snapshots in XDG state with 3-level history (80a6be6)

0.1.0 (2026-02-28)¶

⚠ BREAKING CHANGES¶

  • cli: The -m mode flag is removed; sorting modes must now be passed positionally as ext, alpha, time, or size.

  • cli: The -i option is no longer supported; refinement prompts now run automatically for ext, time, and size modes.

  • cli: remove -i and always prompt refinements (9dc77f2)

  • cli: replace -m with positional mode arguments (dbdc06b)

Features¶

  • cli: prompt for mode on bare smartsort runs (0805942)
  • kind: add 'kind' mode with documents bucket (d387766)
  • kind: make kind the default suggested mode (9d8f683)
  • kind: map archive extensions to archives bucket (4dd057a)
  • kind: map audio extensions to audio bucket (d2c8683)
  • kind: map binary extensions to executables bucket (b36c4b0)
  • kind: map data extensions to data bucket (0a5a264)
  • kind: map disk image extensions to disk-images (b630195)
  • kind: map ebook extensions to ebooks bucket (780b082)
  • kind: map image extensions to images bucket (eee91d3)
  • kind: map source extensions to code bucket (5a8f945)
  • kind: map video extensions to video bucket (0c192b4)

Bug Fixes¶

  • mode: remove unsupported type sorting option (f39ae77)
